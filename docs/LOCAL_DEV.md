# Local development (safe setup)

This repo is an **Expo Router + React Native** app generated by **Rork**.

It does **not** use Python, so `venv` / `conda` won’t help here. For an isolated setup that won’t “mess up” your system, use a version manager (recommended) or a container.

## Option A (recommended): `nvm` + `bun`

1. Install Node with `nvm` (user‑local, doesn’t overwrite system Node):
   - https://github.com/nvm-sh/nvm

2. In this repo:
   ```bash
   nvm install
   nvm use
   ```
   This uses `.nvmrc`.

3. Install Bun (user‑local):
   - https://bun.sh/docs/installation

4. Install deps:
   ```bash
   bun i
   ```

5. Run the app:
   - Web preview:
     ```bash
     bun run start-web
     ```
   - iOS/Android (Expo / Rork dev server):
     ```bash
     bun run start
     ```
     Then press `i` for iOS Simulator or `a` for Android Emulator.

6. Lint:
   ```bash
   bun run lint
   ```

## Environment variables

- Copy `.env.example` to `.env` and adjust if needed:
  ```bash
  cp .env.example .env
  ```
- `EXPO_PUBLIC_RORK_API_BASE_URL` is the base URL for the backend used by tRPC.
  - If you are using `bun run start` / `bun run start-web`, Rork often provides this automatically.
  - If you run on a physical phone, `localhost` will not work; use your computer’s LAN IP (e.g. `http://192.168.1.10:3000`).
